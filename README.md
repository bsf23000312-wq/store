Online Shopping Store — Demo

Files:
- index.html — main storefront
- login.html — login page
- register.html — register page
- css/styles.css — styling
- js/script.js — products, cart, and simple auth using localStorage

Run locally:

1. Open `store/index.html` in your browser (double-click or use "Open File").
2. Use `Register` to create a demo user, then `Login` to sign in.

Notes:
- This is a frontend-only demo. No real payments or servers.
- If you want, I can add server-backed auth, payments, or product management.
